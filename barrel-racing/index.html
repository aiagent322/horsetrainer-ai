The Correct Barrel Racing Progression

The most common mistake: adding speed before the pattern is correct. A horse that can't run it correctly at a lope will only run it more incorrectly at speed. Walk it perfectly before trotting it. Trot it perfectly before loping it. Lope it with control before adding speed.

1
Complete foundation — stop, rate, forward on cue
2
Individual barrel work — walk and trot
3
Full pattern at walk — pocket, turn, departure
4
Trot pattern — correct before adding lope
5
Lope pattern — rate, pocket, turn, drive out
6
Fitness program + progressive speed development
